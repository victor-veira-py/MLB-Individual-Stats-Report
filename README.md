# MLB Individual Stats Executive Report ⚾🐍

Este proyecto automatiza la extracción de estadísticas históricas de jugadores de la **MLB** directamente desde su API oficial, generando reportes en **Excel** con un formato ejecutivo de alta visibilidad.

### ✨ Características Principales:
* **Extracción en Tiempo Real:** Conexión directa con la API oficial de MLB.
* **Cálculos de Carrera:** Generación automática de fila de totales con sumatorias y promedios exactos.
* **Formato Profesional:** Estilo azul empresarial, bordes definidos y promedios estilo .300 (sin el cero inicial).
* **Robustez:** Manejo de datos blindado contra errores de tipo de columna.

### 🚀 Tecnologías utilizadas:
* **Python 3.x**
* **Pandas** (Procesamiento de datos)
* **Requests** (Consumo de API)
* **XlsxWriter** (Diseño de reportes Excel)
